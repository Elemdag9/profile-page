## 1. About the Project

This is a mini frontend project focused on building a clean, semantic, and modern **Profile Page / Profile Card**.

The project is intentionally developed in stages to strengthen core frontend fundamentals. It starts with fully semantic HTML, continues with structured and maintainable CSS, and will later be extended with JavaScript for interactivity.

The goal is not only visual output, but also learning proper structure, layout logic, and a realistic development workflow similar to real-world frontend projects.

## 2. Features

### Implemented

* Fully completed **semantic HTML structure**

  * Uses main, article, figure, section, and nav elements
* All page content is present

  * Profile image placeholder
  * Name and description
  * Statistics section
  * Social media links
* Clean and consistent class naming

  * BEM-inspired and scalable

### In Progress

* CSS styling is **almost finished**
* Layout improvements are ongoing

## 3. Getting Started

This project is a static frontend page and does not require any build tools.

### Prerequisites

* Any modern web browser
* (Optional) A code editor such as VS Code

### Installation

1. Clone the repository
   git clone [https://github.com/Elemdag9/profile-page](https://github.com/Elemdag9/profile-page)

2. Navigate into the project folder
   cd profile-page

3. Open the page
   Open `index.html` directly in your browser
